AASG Geothermal Data: AZGS repository node

Node host name: http://repository.stategeothermaldata.org

Overview

This node on the National Geothermal Data System (NGDS) is operated by the Arizona Geological Survey. To report issues please contact NGDS System Administrator by e-mail at sysadmin at AZGS dot az dot gov or by telephone at 520-770-3500.

This node hosts the document repository for the AASG geothermal data project. This project is supporting compilation of geothermal data from the 50 states by state geological survey organizations. Currently there are NNN documents in the repository. Data acquisition through this project will be ongoing until April, 2014.

The Arizona Geological Survey hosts several other NGDS nodes, including

AZGS NGDS Services node

AZGS AASG geothermal data catalog node

Metadata

Metadata for resources offered by this node can be searched at State Geothermal data resources search, or using the NGDS Portal, which includes metadata for all NGDS resources.

The CSW service for searching resources offered by this node is described by the capabilities document at http://catalog.usgin.org/geoportal/csw?request=GetCapabilities&service=CSW

Metadata for resources offered by this node is also available in a 'web accessible folder' at http://blah.org/metadata. A WAF is a web page with links to NGDS ISO XML metadata documents; see description of NGDS WAF conventions at http://not.writtenYet.org/vaporware.

Capabilities:

This node offers HTTP access to NGDS file-based resources. URLs are encoded in metadata records; no URI template is available for individual resource access.

The node also hosts a web application to allow project participants to create metadata describing resources, upload resource files, bulk load metadata compiled in template tables, and search or browse repository content.
